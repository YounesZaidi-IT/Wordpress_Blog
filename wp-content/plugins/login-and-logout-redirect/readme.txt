This is an easy way to redirect user to specific URL, on login and logout.

== Description ==

This plugin adds extra option in Settings page (Setting>General), for specifying redirect URL for login and logout.

### Features

	* Redirect users instantly on logout.
	* Redirect users instantly on login.
	* Not extra menu option, in the WordPress site. Just head over to Settings>General

== Installation ==

= FOR STANDARD INSTALLATION: =
The simplest way to install is to click on ‘Plugins’ then ‘Add’ and type ‘Login and Logout Redirect’ in the search field. Click install then Activate.

To add redirect URL's visit Settings -> General.

= FOR MANUAL INSTALLATION: =
1. Download the plugin zip file.
Login to your website and go to the Plugins section of your admin panel.
Click the Add New button.
Under Install Plugins, click the Upload link.
Select the plugin zip file from your computer then click the Install Now button.
You should see a message stating that the plugin was installed successfully.
2. Click the Activate Plugin link.
3. Once activated you should check with Settings > General
4. Insert URL for redirect on login.
5. Insert URL for redirect on logout.
6. Save!

= FOR MANUAL INSTALLATION VIA FTP/FILE MANAGER: =
For this method, you will need access to the server or FTP. If you don’t, ask your system administrator to help you out.
1. Download the plugin zip file.
Copy the plugin zip file up to your server and unzip it somewhere on the file system.
Copy the “login-and-logout-redirect” folder into the /wp-content/plugins directory of your WordPress installation.
Login to your website and go to the Plugins section of your admin panel.
Look for “Login and Logout Redirect” and click Activate.
